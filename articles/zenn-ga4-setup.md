## この記事について

いいねは付いてるけど、実際どれくらい読まれてるのか個人的に気になったのでGA4を連携して可視化できるようにしました。

Google Analytics（GA4）を連携すれば、PV数・流入元・どこまで読まれたかなどがわかるようになります。無料で10分くらいで設定できます。

## 設定の流れ

1. GA4で測定IDを取得
2. ZennにIDを登録

これだけです。

## GA4の設定

### アカウント作成

[Google Analytics](https://analytics.google.com/) にアクセスして「測定を開始」。

アカウント名は「Zenn」とかでOK。データ共有設定はそのまま進めて大丈夫です。

### プロパティ作成

| 項目 | 設定 |
|------|------|
| プロパティ名 | Zenn |
| タイムゾーン | 日本（これ忘れずに！） |
| 通貨 | 日本円 |

### ビジネス情報

業種とか規模を聞かれますが、適当でOK。個人利用なので何を選んでも影響ありません。

ビジネス目標は「**ウェブ / アプリのトラフィックの分析**」を選んでおくと、PV確認に便利なレポートが使えます。

### データストリーム作成

プラットフォームは「**ウェブ**」を選択。

| 項目 | 入力 |
|------|------|
| URL | zenn.dev |
| ストリーム名 | Zenn |

:::message
「zenn.devだと全体のデータが入ってくるのでは？」と思うかもしれませんが、大丈夫。Zenn側で自分の記事だけにフィルタリングしてくれます。
:::

### 測定IDをコピー

作成完了すると `G-XXXXXXXXXX` 形式の測定IDが表示されます。これをコピー。

:::message alert
「データ収集が有効になっていません」と警告が出ますが、無視でOK。Zennに登録すれば自動でタグが埋め込まれます。
:::

## Zennに登録

1. [Zenn](https://zenn.dev) にログイン
2. 右上アイコン →「アカウント設定」
3. 「Google Analytics」→「トラッキングIDを設定」
4. 測定IDを貼り付けて保存

これで完了！

## データの確認方法

設定後、24〜48時間でデータが入り始めます。

[Google Analytics](https://analytics.google.com/) で確認できます。

| 見たい内容 | 場所 |
|-----------|------|
| リアルタイムのアクセス | レポート → リアルタイム |
| 記事ごとのPV | レポート → エンゲージメント → ページとスクリーン |
| 流入元 | レポート → 集客 → トラフィック獲得 |

動作確認は、自分の記事を開いてから「リアルタイム」を見てみてください。1人カウントされてればOKです。

## おわりに

一度設定すれば自動でデータが溜まっていくので、記事運営の参考になります。

どの記事が読まれてるか、検索とSNSどっちから来てるか、などがわかると次に何を書くかのヒントにもなりますね。
